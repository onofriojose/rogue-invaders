# Rogue Invaders 🚀

A fast-paced, browser-based space survivor game built with **TypeScript** and **HTML5 Canvas**.

**Rogue Invaders** is a "bullet heaven" roguelite where players pilot customizable ships, fight through infinite sectors of alien swarms, and defeat massive Sector Guardians.

## 🎮 Gameplay Features
* **Endless Progression:** Survive increasingly difficult waves of enemies.
* **Enemy Variety:** Tactical combat against Swarmers (fast/weak), Tanks (slow/heavy), and Ranged units.
* **Boss Battles:** Face off against "Sector Guardians" with unique health bars and mechanics.
* **Economy System:** Collect "Dark Matter" to unlock new ships and upgrades.
* **Visuals:** Neon retro aesthetic with aggressive parallax starfields and screen-shake feedback.
* **Mobile Ready:** Optimized as a PWA (Progressive Web App) for Android installation.

## 🛠️ Tech Stack
* **Language:** TypeScript (Vanilla, no game engines like Phaser/Unity).
* **Rendering:** HTML5 Canvas API (High performance 2D rendering).
* **Build Tool:** Vite.
* **Deployment:** Vercel.
